Usage
--------------------

dcjsの名前でフォルダを作り丸ごとこのプロジェクトを置き、下記をindex.htmlに追加してもらえればそのまま動くと思います。

```html
<div class="col-xs-3 col-sm-2">
  <div class="thumbnail">
    <div class="thumbnail-text-wrapper">
      <a href="dcjs/dcjs.html">
        <img src="dcjs/dcjs.png">
      </a>
      <div class="caption thumbnail-text">
        <strong>dc.js</strong>
      </div>
    </div>
  </div>
</div>
```
